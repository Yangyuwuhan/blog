## 冒泡排序

### 简介

顾名思义，数组里的数据就像水里的气泡一样，大的元素逐渐“浮”到数组的顶端（数组的结尾）。 

### 原理

以升序为例

1. 从头开始，比较第1个和第2个元素，如果第1个 > 第2个，就交换它们
2. 接着比较第2个和第3个，如果第2个 > 第3个，就交换
3. 以此类推，直到比较完倒数第2个和倒数第1个。此时，最大的元素就被放到了最后一位
4. 忽略已经排好的最后一个元素。对剩下的元素重复上述过程，把剩下的数中最大的数放到后面
5. 在只剩下一个元素等待排列时，排序结束

> [!TIP]
> - 在上述操作中，我们进行了 n-1 轮冒泡
> - 实际上，我们可以设置一个标记来记录某一轮是否有元素交换。当某一轮没有元素发生交换时，就说明排序已经完成了

### 代码

```c
void bubble_sort(int arr[], int n) {
    // 第一层循环：控制比较的轮数，最多 n-1 轮
    for (int i = 0; i < n - 1; i++) {
        bool swapped = false;  // 记录本轮是否发生交换
        
        // 第二层循环：相邻比较，每轮最后 i 个元素已经就位
        for (int j = 0; j < n - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                // 交换相邻元素
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
                swapped = true;
            }
        }

        // 如果本轮没有发生任何交换，说明数组已经有序，提前退出
        if (!swapped) {
            break;
        }
    }
}
```

### 特点

1. 空间复杂度: 排序过程只在原数组上进行，仅使用了固定数量的临时变量，故复杂度为 **O(1)**
2. 时间复杂度：
- 最好的情况下，只需要进行一轮冒泡，复杂度为 **O(n)**
- 最坏的情况下，需要进行 n-1 轮冒泡，复杂度为 **O(n²)**
- 平均情况下，冒泡的轮数是 O(n) 级别的，平均复杂度为 **O(n²)**
3. 稳定性：由于相等的两个元素没有发生交换，故冒泡排序是 **稳定的**

## 鸡尾酒排序(双向冒泡排序)

### 简介

- 基本原理与**冒泡排序**相同，区别在于，鸡尾酒排序会在每一轮“冒泡”后交换冒泡的方向
- 对于错误元素集中在两端的情况下，双向冒泡排序会普通冒泡排序更快。排跃“n/2”轮即可
- 由于优化有限，且实现较为复杂，大部分情况下鸡尾酒排序**不被使用**

### 代码

```c
// 辅助交换
void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}
// 鸡尾酒排序
void cocktail_sort(int arr[], int n) {
    int left = 0;
    int right = n - 1;
    bool swapped = true;
    while (swapped) {
        swapped = false;
        // 正向扫描：从左到右，将最大值移到右边
        for (int i = left; i < right; i++) {
            if (arr[i] > arr[i + 1]) {
                swap(&arr[i], &arr[i + 1]);
                swapped = true;
            }
        }
        if (!swapped) break;
        right--;// 缩小右边界
        // 反向扫描：从右到左，将最小值移到左边
        for (int i = right; i > left; i--) {
            if (arr[i] < arr[i - 1]) {
                swap(&arr[i], &arr[i - 1]);
                swapped = true;
            }
        }
        left++;// 缩小左边界
    }
}
```
